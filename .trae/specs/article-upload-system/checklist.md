# AI知识库 - 文章上传与管理系统 - 验证清单

## 后端API验证
- [ ] 文章上传API能成功接收文章数据并返回201状态码
- [ ] 上传的文章默认状态为pending
- [ ] 文章上传API能验证必填字段
- [ ] 管理员登录API能验证管理员凭据并返回JWT token
- [ ] 登录失败时返回401状态码
- [ ] 受保护的接口需要验证JWT token
- [ ] 管理员文章列表API能返回所有文章
- [ ] 文章状态更新API能更新文章状态
- [ ] 文章编辑API能编辑文章
- [ ] 文章删除API能删除文章

## 前端功能验证
- [ ] 文章上传表单能成功提交到后端API
- [ ] 文章上传表单界面美观易用，有清晰的错误提示
- [ ] 文章上传后显示成功或失败的反馈
- [ ] 管理员登录表单能成功提交到后端API
- [ ] 管理员登录成功后跳转到管理界面
- [ ] 管理员登录失败时显示错误消息
- [ ] 文章管理界面清晰易用，操作按钮明显
- [ ] 文章管理界面能成功获取和显示所有文章
- [ ] 文章管理界面能成功执行审核、编辑和删除操作
- [ ] 文章管理界面操作后显示成功或失败的反馈
- [ ] 前台只显示status为published的文章
- [ ] 待审核和已拒绝的文章不显示在前台

## 性能和用户体验验证
- [ ] 文章上传和管理操作响应时间不超过2秒
- [ ] 页面加载时间不超过3秒
- [ ] 界面美观易用，与现有UI风格一致
- [ ] 有清晰的错误提示和用户反馈
- [ ] 响应式设计，支持不同设备

## 安全性验证
- [ ] 管理员登录使用安全的身份验证机制
- [ ] 密码使用加密存储
- [ ] 敏感操作需要权限验证
- [ ] 只有管理员能访问管理功能
