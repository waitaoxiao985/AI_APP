# 登录状态修复 - 验证清单

- [x] 注册成功后，token 应该被保存到 localStorage
- [x] 注册成功后，用户应该被导航到发现页面
- [x] 登录成功后，token 应该被保存到 localStorage
- [x] 登录成功后，用户应该被导航到发现页面
- [x] 用户已登录时，个人中心页面应该显示用户信息
- [x] 用户未登录时，个人中心页面应该显示未登录状态
- [x] 退出登录后，token 应该从 localStorage 中清除
- [x] 退出登录后，个人中心页面应该显示未登录状态
- [x] 所有功能应该在清除浏览器缓存后仍然正常工作
- [x] 所有功能应该在不同浏览器中正常工作
